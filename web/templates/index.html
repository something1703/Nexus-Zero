Under the Hood

Built on Google Cloud

Fully serverless architecture. Every component runs on Cloud Run with Cloud SQL for state and Cloud Logging for observability.

ü§ñ
MCP Protocol

Each agent exposes its capabilities as MCP tools ‚Äî structured, discoverable, and composable. The bridge layer translates between MCP tool calls and GCP APIs.

üîó
Archestra A2A

Agent-to-agent communication powered by the Archestra Hub. Sentinel orchestrates sub-agents (Detective, Historian, Mediator, Executor) for coordinated incident response.

Cloud Run

8 serverless services deployed to Cloud Run ‚Äî 5 agents, 3 demo services. Auto-scaling from zero, sub-second cold starts, pay only when processing.

PostgreSQL 18 stores incidents, audit logs, playbooks, and service metadata. Full audit trail of every agent action for explainability and compliance.

Deploy on Your GCP Project

Nexus-Zero is fully portable. One command deploys the entire platform ‚Äî 5 AI agents, demo service, MCP bridge, and dashboard ‚Äî to any Google Cloud project in under 15 minutes.

  1. 1 Clone the repo & run ./deploy.sh
  2. 2 Provide your GCP project ID & Gemini API key
  3. 3 Script creates Cloud SQL, deploys 8 Cloud Run services
  4. 4 Register agents in Archestra Hub (5 SSE endpoints)
  5. 5 Open your dashboard & inject chaos
üí∞ ~$0.30/day ‚Äî Cloud SQL is the only always-on cost. All Cloud Run services scale to zero when idle. Full teardown with ./teardown.sh.
